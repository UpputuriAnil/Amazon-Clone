# Amazon Clone

A responsive Amazon clone built with HTML and CSS.

## Features
- Responsive navigation with search
- Product category grid
- Modern UI design

## Tech Stack
- HTML5
- CSS3
- Font Awesome

## Quick Start
```bash
git clone https://github.com/UpputuriAnil/Amazon-Clone.git
cd Amazon-Clone
```
Visit `http://localhost:8000`
